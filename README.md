# おみくじアプリ 🎋

おみくじを引くWebアプリです。AIで生成した動画・音声素材を使ったアニメーション付きです。

## デモ

[Vercelデプロイ後にURLを記載]

## 機能

- 🎋 おみくじ箱シャカシャカアニメーション（AI動画生成）
- 🎴 棒が出るアニメーション（AI動画生成）
- ✨ 大吉・吉・中吉・小吉・末吉・凶の6段階
- 🎵 和風BGM・SE（AI音声生成）
- 📱 スマホ対応（9:16縦型）

## 技術スタック

- Next.js 14 (App Router)
- TypeScript
- AI生成素材（Kling動画生成 / ElevenLabs SE / Gemini TTS BGM）

## 開発

```bash
npm install
npm run dev
```
